# Screenshot Capture Guide

Take these screenshots from http://192.168.13.20:8101 and save them as PNG files in this folder.

**Tips:**
- Use browser at 1280x800 or similar widescreen resolution
- Dark theme looks better on GitHub (dark background README)
- Use `Ctrl+Shift+S` in Firefox or DevTools to capture full page
- Crop to remove browser chrome
